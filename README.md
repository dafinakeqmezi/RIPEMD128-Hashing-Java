# RIPEMD128-Hashing-Java

Ky projekt demonstron përdorimin e algoritmit të hashimit RIPEMD-128 duke përdorur librarinë **BouncyCastle**.

## Si funksionon
- Merr një tekst nga përdoruesi
- Gjeneron hash-in RIPEMD128
- Shfaq rezultatin në format hexadecimal

## Shembull
Input: `Siguria e informacionit`  
Output: `b8a29e98e1a07b2e0cbde367b4b21af5`
